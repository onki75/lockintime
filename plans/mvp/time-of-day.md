# 使用時刻制限 + 曜日別スケジュール

特定の曜日・時間帯だけサイトをブロックする。
「平日の仕事中だけSNSをブロック」のようなユースケースに対応。

---

## 機能

- 曜日と時間帯を指定してブロックスケジュールを設定
- 1つのルールに複数スケジュールを設定可能（例: 平日9-18時 + 土曜10-15時）
- スケジュール範囲内 → ブロック、範囲外 → アクセス可能
- 平日/休日プリセットで簡単に曜日選択

---

## Chrome API

- `chrome.alarms` — 1分ごとにスケジュールチェック
- `declarativeNetRequest` — スケジュール範囲内/外でルール有効化/無効化

### UI
- サイト追加ダイアログに使用時刻制限の選択肢を追加
  - 曜日選択（平日/休日プリセット + 個別チェックボックス）

- 設定ページ: 編集ダイアログでスケジュール変更

## 画面設計

### サイト追加ダイアログ（使用時刻制限選択時）

```
│ ○ 使用時刻制限                        │
│     曜日: [●月 ●火 ●水 ●木 ●金 ○土 ○日] │
│           [平日] [休日] [毎日]          │
│     時間: [09:00] 〜 [18:00]          │
│     [+ スケジュール追加]               │
```

### ポップアップ表示

```
│ 🕐 x.com       平日9:00〜18:00  │
```

### ブロックページメッセージ

```
│ このサイトはブロック中です            │
│      x.com                         │
│ 18:00以降にアクセスできます           │
```
