## 概要（短い説明 / 132 文字以内）
日本語特化のサイトブロッカー。YouTube・X・漫画サイトをブロックして集中時間を確保。PC のスクリーンタイムも管理できます。

## 詳細な説明（Description）
LockInTime は「ついつい見てしまう」サイトをブロックして、集中したい時間を守るための拡張機能です。日本語ユーザー向けに作られています。

【主な機能】
■ 多彩なブロック方式
- フルブロック: 指定サイトを完全に遮断
- 時間帯ブロック: 「9〜18時だけ」など時間で制限
- 1日の回数・利用時間制限: 「1日3回まで」「合計30分まで」
- クールダウン / 遅延: 開く前に待ち時間を挟んでドカ見を防止
- 位置情報ブロック: 「会社にいるときだけ」など場所で制限

■ スクリーンタイム管理
PC での利用時間を可視化。スマホだけでなく PC の使いすぎも把握できます。

■ 続けたくなる仕組み
連続達成日数（ストリーク）やマイルストーンで、集中の習慣化をサポート。

【プライバシー】
設定やスクリーンタイムの記録は、原則すべてお使いの端末内に保存され、外部に送信されません。閲覧履歴の収集やトラッキングは行いません。

### データ利用の開示（Data usage）回答指針
現行 v0.1.0 は外部送信が一切ないため、**どのデータ種別もチェックしない（=収集しない）**。
- 個人を特定できる情報 / 健康 / 財務 / 認証 / コミュニケーション / **位置情報** / ウェブ履歴 / ユーザーアクティビティ / ウェブサイトコンテンツ: **すべてチェックしない**
  - 位置情報は端末内のルール判定のみで外部送信しないため「収集」に該当しない（geolocation 権限の説明欄に明記済み）
- 以下3点のチェックボックスはすべてチェック（すべて該当）:
  1. 承認された用途以外でユーザーデータを販売・転送しない
  2. 信用力判断・融資目的でユーザーデータを使用・転送しない
  3. ユーザーデータの用途は単一目的に関連している
- プライバシーポリシー URL は提供する（収集しなくても掲載は可・推奨）
