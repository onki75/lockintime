## STEP 2. ストアの掲載情報（Store listing）

**アイテム名:**
```
LockInTime - サイトブロッカーで集中モード
```

**概要（短い説明・132文字以内）:**
```
日本語特化のサイトブロッカー。YouTube・X・漫画サイトをブロックして集中時間を確保。PC のスクリーンタイムも管理できます。
```

**詳細な説明:**
```
LockInTime は「ついつい見てしまう」サイトをブロックして、集中したい時間を守るための拡張機能です。日本語ユーザー向けに作られています。

【主な機能】
■ 多彩なブロック方式
- フルブロック: 指定サイトを完全に遮断
- 時間帯ブロック: 「9〜18時だけ」など時間で制限
- 1日の回数・利用時間制限: 「1日3回まで」「合計30分まで」
- クールダウン / 遅延: 開く前に待ち時間を挟んでドカ見を防止
- 位置情報ブロック: 「会社にいるときだけ」など場所で制限

■ スクリーンタイム管理
PC での利用時間を可視化。スマホだけでなく PC の使いすぎも把握できます。

■ 続けたくなる仕組み
連続達成日数（ストリーク）やマイルストーンで、集中の習慣化をサポート。

【プライバシー】
設定やスクリーンタイムの記録は、原則すべてお使いの端末内に保存され、外部に送信されません。閲覧履歴の収集やトラッキングは行いません。
```

**カテゴリ:** 仕事効率化（Productivity）
**言語:** 日本語

## STEP 3. プライバシー（Privacy practices）タブ

**単一目的の説明:**
```
LockInTime は、ユーザーが指定したウェブサイトをブロック・制限し、PC のスクリーンタイムを可視化することで集中を支援する拡張機能です。すべての機能はこの単一目的（集中のためのサイト制限とスクリーンタイム管理）に紐づいています。
```

**各権限の用途**（→ `permission-justifications.md` から各欄に転記）
storage / alarms / declarativeNetRequest / tabs / webNavigation / geolocation / offscreen / favicon / host(`<all_urls>`) の9項目。

**リモートコード:** 「使用していない」を選択

**データ利用の開示:**
- メールアドレス → **収集する**（用途: アプリ機能=ライセンス管理／有料時のみ）
- 位置情報 → 端末内処理のみ・外部送信なし（用途: アプリ機能=位置ベース制限）
- 閲覧履歴・ウェブ閲覧アクティビティ → **収集しない**
- 3つの同意チェックボックス → すべてチェック
