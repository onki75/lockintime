# 設定ページ（Options）

メインの管理画面。フルページ（options.html）で表示。

---

## セクション構成

上から順に:

1. **ヘッダー** — ロゴ + プラン情報
2. **ブロックリスト** — ルール一覧 + 追加ボタン
3. **アダルトフィルター** — ON/OFFトグル
4. **プリセット** — カテゴリ別一括追加
5. **データ管理** — エクスポート/インポート

---

## 2. ブロックリスト

### ルール一覧

各ルールはカード形式で表示。SiteRuleとGroupRuleで見た目を分ける。

```
■ ブロックリスト

┌──────────────────────────────────────────────────────────┐
│ ⛔ youtube.com                                           │
│ 完全ブロック                              [ON/OFF] [🗑]  │
└──────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│ 🕐 x.com                                                │
│ 使用時刻制限: 平日 9:00〜18:00            [ON/OFF] [🗑]  │
└──────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────┐
│ 📁 SNS  (6サイト)                          [ON/OFF] [🗑] │
│ 完全ブロック                                              │
│ x.com, twitter.com, instagram.com, facebook.com...      │
│                                            [▼ 展開]     │
└──────────────────────────────────────────────────────────┘

[+ サイトを追加]  [+ グループを作成]
```

### サイト追加ダイアログ

```
┌──────────────────────────────────────────┐
│ サイトを追加                              │
├──────────────────────────────────────────┤
│ ドメイン                                  │
│ [youtube.com                          ]  │
│                                          │
│ 制限タイプ                                │
│ ● 完全ブロック                            │
│ ○ 使用時刻制限                            │
│     曜日: [月][火][水][木][金][土][日]     │
│           [平日] [休日] [毎日]            │
│     時間: [09:00] 〜 [18:00]             │
│     [+ スケジュール追加]                   │
│                                          │
│ ── Pro機能（🔒） ──                       │
│ ○ 使用回数制限   1日 [3] 回まで           │
│ ○ 使用時間制限   1日 [30] 分まで          │
│ ○ クールダウン   [30] 分間隔              │
│ ○ 遅延アクセス   [10] 秒待機              │
│ ○ 位置情報制限   [職場 ▼]                │
│                                          │
│          [キャンセル] [追加]              │
└──────────────────────────────────────────┘
```

MVP段階: 完全ブロック + 使用時刻制限のみ選択可能。
Pro機能はグレーアウト + 🔒アイコン + 「Proプランで利用可能」ツールチップ。

## 4. プリセット

```
■ プリセットから追加

┌──────────────────────────────────────────────────────────┐
│ [SNS (6)]  [動画 (5)]  [ニュース (4)]                     │
│ [漫画・小説 (6)]  [掲示板・まとめ (4)]                     │
└──────────────────────────────────────────────────────────┘

Free: クリック → サイト選択UI（5件上限内で個別選択）
Pro/トライアル: クリック → グループとして一括追加
```

## コンポーネント分割

```
src/options/
├── Options.tsx          # ルート（セクション配置）
├── components/
│   ├── Header.tsx       # ヘッダー + プラン情報
│   ├── RuleList.tsx     # ブロックリスト一覧
│   ├── RuleCard.tsx     # SiteRule 1件のカード
│   ├── GroupCard.tsx    # GroupRule 1件のカード（展開可能）
│   ├── AddSiteDialog.tsx    # サイト追加ダイアログ
│   ├── PresetSection.tsx    # プリセットセクション
│   ├── PresetPicker.tsx     # Free用プリセット選択UI
│   ├── AdultFilter.tsx      # アダルトフィルタートグル
│   └── DataManagement.tsx   # エクスポート/インポート
src/components/
├── Toggle.tsx           # 共通トグルスイッチ
├── Button.tsx           # 共通ボタン
├── Dialog.tsx           # 共通モーダル
└── Badge.tsx            # 制限タイプバッジ
```
