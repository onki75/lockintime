# 一時解除（バイパス）

ブロック中のサイトに一時的にアクセスを許可する。
「今だけ5分間YouTubeを見たい」のようなユースケースに対応。

---

## 機能

- ブロックページから「一時的にアクセスする」を選択
- 解除時間を選択
- 解除中はブロックが無効化され、通常通りサイトにアクセスできる
- 設定時間経過後に自動でブロック復帰
- ポップアップに解除中のサイトと残り時間を表示

### プラン別の機能差

| | Free | Pro / Cloud |
|---|---|---|
| 解除時間 | **5分のみ** | 5分 / 15分 / 30分 / カスタム |

Free版はMVPに含める（ブロックページに「5分だけアクセスする」ボタン）。
Pro版はP1で解除時間の選択肢を拡張する。

---

## Chrome API

- `chrome.alarms` — 解除終了タイマー
- `declarativeNetRequest` — 一時的にルールを無効化/再有効化

---

## タスク

### Background
- 一時解除の状態管理（`chrome.storage.local` に保存）
  - `{ ruleId, expiresAt }` の配列で管理
- `chrome.alarms` で解除終了時にルールを再有効化
- Service Worker再起動時に未期限の一時解除を復元

### UI
- ブロックページに「一時的にアクセスする」ボタンを追加
- 解除時間の選択ダイアログ
- ポップアップに解除中サイトの表示（残り時間付き）
