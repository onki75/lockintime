## プラン詳細

### Free（無料）
ログイン不要で使える基本プラン。

- 完全ブロック + 使用時刻制限（曜日別スケジュール含む）
- **単体ルール: 5サイトまで**
- プリセットからは個別サイトを選んで単体ルールとして追加（5件上限内）
- グループ機能なし
- ブロックページ表示
- ストリーク（数値のみ、ヒートマップはPro）
- デイリーチャレンジ（1つ/日、ブロンズのみ）
- レスキューパス（トライアル中に2枚配布、以降は補充なし）
- セレブレーション演出（簡易版）

**目的**: まず使ってもらい、価値を体験してもらう。ゲーミフィケーションの味見でPro転換を促す。

**初回トライアル**: インストールから**7日間はPro全機能を無料体験**可能（グループ含む）。カード登録不要。**レスキューパス2枚を即時配布**（トライアル中に使い方を体験）。期間終了後に自動でFreeに移行（ダウングレードUIを表示）。

### Pro（月額480円〜 / 年額3,800円〜 / 買い切り7,800円〜）
全制限タイプ + グループ管理が使えるローカル完結プラン。価格はPhaseにより変動。

- **全7制限タイプ解放**
- 単体ルール: **無制限**
- **グループ作成**（カスタム + プリセットグループ）
- **グループ単位のON/OFF**
- ストリーク機能（ヒートマップ + 数値 + 修復 + フリーズ）
- レスキューパス（一時解除 + フリーズ + マスコット餌）
- デイリーチャレンジ（3つ/日）
- セレブレーション演出（フル版）
- マスコット育成
- 一時解除（バイパス）
- ロックモード（パスワード保護）
- 優先サポート

**買い切りについて**:
- 買い切り後に追加されるPro機能は利用可能

## 3つの割引制度

### 1. トライアル特別価格（Phase D到達後のみ）

新規ユーザーの7日間トライアル期間中に契約すると特別価格で購入可能。
Phase A〜Cでは不要（価格自体が十分安い。「今の価格は今だけ」で訴求）。

| | 通常価格（D） | トライアル特別価格 |
|---|---|---|
| Pro月額 | 1,480円 | **680円**（Phase B価格） |
| Pro年額 | 12,800円 | **5,800円** |
| Pro買い切り | 19,800円 | **9,800円** |

**条件:**
- 7日間のトライアル期間中のみ
- Googleログイン必須（濫用防止）
- 1アカウントにつき1回限り
- 特別価格でのgrandfatheringあり（ずっとその価格）

### 2. 解約引き止め割引（全Phase共通）

解約フロー内で「続けませんか？」とオファー。

- **現在の支払額の50%OFF × 3ヶ月間**
- 3ヶ月後に元のgrandfathering価格に戻る
- **1ユーザーにつき1回限り**（2回目の解約時は割引なし）

| Phase | 通常価格 | 引き止め価格（3ヶ月限定） |
|-------|---------|------------------------|
| A契約者 | 480円/月 | **240円/月** |
| B契約者 | 680円/月 | **340円/月** |
| C契約者 | 980円/月 | **490円/月** |
| D契約者 | 1,480円/月 | **740円/月** |

**解約引き止めUI:**

```
┌──────────────────────────────────────┐
│ 本当に解約しますか？                   │
│                                      │
│ 🎁 特別オファー                       │
│ 今なら3ヶ月間 半額で継続できます       │
│                                      │
│ 現在: 480円/月 → 240円/月（3ヶ月間） │
│ ※ 3ヶ月後に通常価格に戻ります         │
│                                      │
│  [半額で継続する]  [解約を続ける]      │
└──────────────────────────────────────┘
```

### 3. Grandfathering

契約時のPhase価格をずっと維持。値上げされても既存ユーザーは旧価格。

**効果:**
- 「今やめたら次は高い」→ 解約抑止
- Phase Aで480円のユーザーは、Phase Dで1,480円になっても480円のまま
- 早期ユーザーへの感謝 + 口コミ促進

### 割引の序列

```
トライアル特別(680円) < 解約引き止め(740円) < 通常D(1,480円)
```

トライアル特別が最安。解約引き止めは通常の半額。序列が保たれている。

## 決済

### Stripe
- 拡張の「Proにアップグレード」ボタン → **Stripe Checkoutページ**をブラウザで開く
- 決済完了 → Stripe Webhook → Firestoreにライセンス反映
- 拡張がFirestore経由でプラン変更を検知
- サブスク管理（プラン変更・解約）もStripeのカスタマーポータルで完結

### 決済失敗時の挙動
- Stripeが自動リトライ（最大3回、数日間隔）
- 最後のリトライ失敗から**7日間の猶予期間**
- 猶予期間中はPro機能を維持しつつ「お支払いに問題があります」バナーを表示
- 7日経過後にFreeにダウングレード（ダウングレード選択UIを表示）

### セキュリティ注意事項
- Stripeのエラーレスポンスに含まれる機密情報を**ログに出力しない**
- 決済はStripe Checkoutページで完結し、拡張側でカード情報を一切扱わない（PCI-DSS準拠）

### 価格設計の原則
- 価格はFirestoreで管理し、**拡張のコードにハードコードしない**
- Stripeの価格ID（price_xxx）をFirestoreに保存し、拡張が参照する
- 価格変更時はFirestore + Stripeを更新するだけで即反映

## ダウングレード時のデータ処理

Pro → Free へのダウングレード時、データは一切削除しない。

### ルール
- **Pro限定タイプ**（回数・時間・クールダウン・遅延・位置情報）→ 自動ロック（動作停止、鍵アイコン表示）
- **Free対応タイプが5件超** → ユーザーに「どの5件を有効にするか」選択UIを表示
- 選択されなかったルールはロック状態で保持
- 再アップグレードで全ルール復活

## トライアル管理

段階的に管理レベルを強化するハイブリッド方式。

### Phase A〜C（MVP〜P1初期）: ローカル管理

**仕組み:**
- `chrome.storage.local` に `trialStartDate` を保存（ミリ秒単位のUnixタイムスタンプ）
- `chrome.runtime.onInstalled` でインストール時に記録（`reason === 'install'` のみ）
- `Date.now() < trialStartDate + 7 * 24 * 60 * 60 * 1000` → Pro機能有効
- 期限切れ → Freeに移行 + ダウングレードUI表示

### P1以降（Google認証導入後）: Firestore管理

**仕組み:**
- Googleログイン済みユーザー → Firestoreで管理（再インストールでもリセット不可）
- 未ログインユーザー → ローカル管理

### Phase D（トライアル割引導入後）: ログイン必須

**仕組み:**
- トライアル自体はローカルでも開始可能
- **トライアル割引価格での購入にはGoogleログイン必須**
- 1アカウントにつきトライアル割引は1回のみ
