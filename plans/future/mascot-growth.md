# マスコット育成 + レスキューパスシステム

## 概要

ストリーク継続で獲得する「レスキューパス」を、一時解除に使うかマスコットの餌にするか選べる。
マスコットに餌を与えると成長・進化する。たまごっち的なゲーミフィケーション。

---

## レスキューパス

- 7日間連続でストリーク達成 → パス1枚獲得
- パスの使い道:
  - **一時解除に使う**: 15分間ブロック解除。ストリークは「一時解除」扱い（失敗にはならない）
  - **マスコットの餌にする**: マスコットが成長する

---

## マスコット成長

| レベル | 必要な餌 | 見た目 | 累計日数目安 |
|--------|---------|--------|------------|
| 🥚 たまご | 0 | 初期状態 | 0日 |
| 🐣 ひよこ | 3個 | 小さい盾キャラ | 21日 |
| 🛡 見習い | 7個 | 通常の盾キャラ | 49日 |
| ⚔️ 戦士 | 15個 | 装飾付き盾 | 105日 |
| 👑 マスター | 30個 | 王冠付き盾 | 210日 |

- パスを一時解除に使うとマスコットが悲しむ（退化はしない）
- マスコットはポップアップのカレンダー上部に常駐表示
- レベルアップ時に演出（紙吹雪等）

---

## 一時解除フロー

1. ポップアップのストリークカレンダーで**今日のセル**をタップ
2. **反省カード**が表示:
   - 「本当にアクセスしますか？」
   - 現在のストリーク日数
   - 「ストリークに一時解除として記録されます」
   - 理由選択（仕事で必要 / 緊急連絡 / その他）
3. **3秒長押し**で確定
4. 15分間ブロック解除 → 自動復帰

### レスキューパスあり vs なし

| パス | ストリーク記録 | マスコットの反応 |
|------|-------------|----------------|
| パス消費して解除 | 「一時解除」（黄色） | 少し悲しむ |
| パスなしで解除 | 「失敗」（赤） | かなり悲しむ |
| 解除しない | 「成功」（緑） | 笑顔 |

## 差別化ポイント

- 競合のサイトブロッカーにはマスコット育成要素がない
- 「ブロックを続ける = マスコットが育つ」というポジティブな動機付け
- レスキューパスの「使うか育てるか」のジレンマが自制を強化
- SNSでの共有ネタにもなる（「マスコットが戦士に進化した！」）

---

## フェーズ

- **P2**: レスキューパス基本実装（一時解除のみ）
- **P2+**: マスコット育成（餌・レベル・見た目変化）
- **Future**: マスコットのカスタマイズ（色、アクセサリー等）

## UIモード切替

ビジネス向けにマスコットなしの「シンプルモード」を用意する。

| | シンプルモード | マスコットモード |
|---|---|---|
| ポップアップ | ストリーク数値 + カレンダー（色のみ） | マスコット表示 + 育成UI + カレンダー（スタンプ） |
| ブロックページ | 名言のみ | マスコットの表情 + 名言 |
| 一時解除 | 反省カード + 長押し確定 | 同じ + マスコットの反応 |
| レスキューパス | パス消費のみ | パス消費 or 餌として使用 |
| 設定 | 表示設定タブで切替 | デフォルト |

- 設定ページの「表示設定」タブで切替: ● マスコットモード / ○ シンプルモード
- シンプルモードでもレスキューパスは機能する（餌オプションがないだけ）
- ビジネス向け・シンプル好みのユーザーに対応
- マスコットモードがデフォルト

### オンボーディングでのモード選択

マスコット登場後（Step 1）に選択肢を提示:

```
┌────────────────────────────────────┐
│      🐣 はじめまして！              │
│                                    │
│ どちらのモードで使いますか？         │
│                                    │
│ ┌──────────────┐ ┌──────────────┐ │
│ │ 🐣 マスコット │ │ 🛡 シンプル  │ │
│ │              │ │              │ │
│ │ キャラと一緒に│ │ すっきりした  │ │
│ │ 楽しく続ける  │ │ ビジネス向け  │ │
│ │              │ │              │ │
│ │ (おすすめ)    │ │              │ │
│ └──────────────┘ └──────────────┘ │
│                                    │
│ ※ あとから設定で変更できます        │
└────────────────────────────────────┘
```

- マスコットモードが左側（おすすめラベル付き）
- シンプルモードは控えめに表示
- 選択後、マスコットモードならStep 2でマスコットが案内を続行
- シンプルモードならマスコット非表示でStep 2に進む
- 「あとから設定で変更できます」の注記で安心感
