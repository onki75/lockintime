# PCでスクリーンタイムを設定する方法【2026年版】Mac・Windows・Chrome対応

「気づいたら1時間、Twitterを見ていた」

リモートワーク中、レポートの締め切り前、副業の作業中。
PCの前にいると、スマホ以上にSNSやYouTubeの誘惑が強い。

iPhoneには「スクリーンタイム」があって、アプリの使用時間を可視化して制限できる。
でもPCには？ —— 実は、ほとんどの人が**PCだけ無法地帯**のまま放置しています。

この記事では、PCでスクリーンタイムを設定する方法を**OS標準機能・Chrome拡張・デスクトップアプリ**の3つに分けて、メリット・デメリットを正直に比較します。

先に結論を書いておくと、「何でブロックするか」より**「どう続けるか」が本当の課題**です。

---

## そもそも、なぜPCが問題なのか

スマホ依存はよく話題になります。Job総研の2024年調査では**20代の86.5%**が「スマホの使いすぎをやめたい」と回答しています。

でも本当に怖いのはPCです。

スマホなら「スクリーンタイム見たら4時間使ってた、やばい」と気づける。
PCは？ **そもそも計測すらされていない。**

仕事のふりをしてブラウザのタブでTwitterを開いている時間。
「調べ物」と称してYouTubeに吸い込まれる時間。

これが可視化されたら、ぞっとする人は多いはずです。

## 方法1: OS標準機能（無料だが穴だらけ）

### Mac：スクリーンタイム

iPhoneと同じ「スクリーンタイム」がMacにも搭載されています。

**できること：**
- アプリごとの使用時間制限
- 時間帯を指定して「休止時間」を設定
- Webサイトの制限（成人向けサイトのフィルタリング等）

**問題：**
- **Safariでしか機能しない。** Chromeを使っている人には意味がない
- パスコードを自分で設定→自分で解除できてしまう
- サイトごとの「1日○分まで」という制限はできない

Chromeがメインブラウザの人（大多数）には、残念ながら使えません。

### Windows：ファミリーセーフティ

Windows 11の設定にある機能です。

**できること：**
- アプリごとの使用時間制限
- Webフィルタリング

**問題：**
- **子供向けの機能。** 自分自身を制限する設計になっていない
- WebフィルタリングはEdgeのみ。Chromeは対象外
- Microsoftアカウントのファミリーグループを作る必要があり、設定が面倒

ちなみにWindows 11の「集中モード」は通知を止めるだけ。サイトブロック機能はゼロです。

### 正直な結論

**Chromeユーザーには、OS標準機能だけではどうにもならない。**
Mac・Windows共に、標準ブラウザ以外のサイト制限には対応していません。

## 方法2: Chrome拡張機能（手軽だが抜け道がある）

ここからが本題。Chromeをメインで使っているなら、拡張機能が最も現実的です。

### BlockSite — 知名度No.1、でも無料の罠

ユーザー数100万超。「サイトブロッカー」で検索すると最初に出てきます。

ブロックリストの作成、スケジュール設定、パスワード保護と一通りの機能が揃っている。スマホアプリとの同期もできる。

**ただし、無料版は3サイトまでしかブロックできません。**

Twitter、YouTube、Instagramの3つでもう上限。ニュースサイトやRedditも追加したければ月額約600円の有料版が必要です。レビューでも「無料の制限が厳しすぎる」という不満が最も多い。

### StayFocusd — 無料で使える定番

「1日にTwitterは10分まで」のように、サイトごとの使用時間制限ができる。全機能無料。

目玉は**「Nuclear Option」**。起動すると、設定した時間が経つまで絶対にブロックを解除できない。本気で集中したいときに便利。

ただしUIが2010年代から変わっていない古さがあり、クラウド同期もなし。Chrome限定。

### LeechBlock NG — 玄人向けの高機能

オープンソースのブロッカー。複数のブロックセットを作成でき、「平日9-18時はSNS全ブロック」「土日は2時間まで」のような複雑なルールが組める。Chrome / Firefox両対応。

ただし設定画面が複雑で、初心者にはハードルが高い。

### Chrome拡張の比較

| | BlockSite | StayFocusd | LeechBlock NG |
|---|---|---|---|
| 無料の範囲 | **3サイトまで** | 全機能 | 全機能 |
| 時間制限 | ○（有料） | ○ | ○ |
| 解除防止 | パスワード | Nuclear Option | ロックダウン |
| 使いやすさ | ◎ | ○ | △（複雑） |

**Chrome拡張の共通の弱点が1つあります。**

拡張機能の管理画面（`chrome://extensions`）から無効化・削除すれば、全てのブロックが解除されます。「本気で止めたい」人にとって、これは致命的です。

意志の力が弱い瞬間——深夜、締め切り前、疲れている時——に、この抜け道を使ってしまう。

## 方法3: デスクトップアプリ（最も強力）

「自分の意志を信用できない」人には、デスクトップアプリが選択肢になります。

### Freedom — 全デバイスを一括ブロック

Mac / Windows / iOS / Android / Chrome に対応。PCでブロック中はスマホでもアクセスできない、という「逃げ場をなくす」設計。

「Locked Mode」を有効にすると、セッション中はブロック解除不可能。年額約5,000円（月換算420円）。無料プランなし。

### Cold Turkey — 最も厳格。本当に解除できない

Pro版（約5,500円・買い切り）は、一度ブロックを開始すると**アンインストールしても、PCを再起動しても解除されない**。

「やりすぎ」と思うかもしれないが、レビューを見ると「ここまでしないと自分は止められない」という人が一定数いる。

ただし、誤って24時間ブロックを設定してしまい仕事に支障が出た、という事故もある。

### SelfControl — Mac専用・無料で最強

ブロックリストに入れたサイトへのアクセスを、指定時間だけ完全遮断。再起動しても解除されない。無料。

ただしMac専用で、スケジュール機能もない。「タイマー + ブロックリスト」だけの潔いシンプルさ。

### デスクトップアプリの比較

| | Freedom | Cold Turkey Pro | SelfControl |
|---|---|---|---|
| 価格 | 年5,000円 | 5,500円買い切り | 無料 |
| 対応OS | 全OS | Win / Mac | Macのみ |
| 解除不可能度 | 高 | **最高** | **最高** |
| スケジュール | ○ | ○ | × |

## どのツールを選ぶべきか

ここまで読んで、「じゃあCold TurkeyかFreedomを入れればいいのか」と思ったかもしれません。

**入れてみてください。おそらく1週間で使わなくなります。**

これは私の推測ではなく、サイトブロッカー全般に共通する現象です。BlockSiteのレビューでも、Cold Turkeyのフォーラムでも、同じ声が繰り返されている。

> 「最初は効果があった。でも2週間で設定すらしなくなった」

理由はシンプルで、**ブロッカーは「遮断する」ことしかできない**からです。

遮断される → ストレスが溜まる → 「今日くらいいいか」と解除する → 元に戻る

Duolingoを想像してみてください。もし「1日サボったらアプリが使えなくなる」だけの設計だったら、誰も続けない。**ストリーク（連続記録）の表示、達成時のお祝い演出、「あと1日で記録更新」の通知**。続けたくなる仕組みがあるから、続く。

サイトブロッカーに足りないのは、強制力ではなく**「続けたくなる理由」**です。
