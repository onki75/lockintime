# 利用統計・インサイト

ブロック成功回数や節約時間を可視化する。
DailyStatsのデータを活用して、ユーザーのモチベーションを維持する。

---

## 機能

- **週次サマリー**: 今週のブロック成功回数、節約時間、最もブロックしたサイト
- **月次トレンド**: 月ごとのブロック回数推移グラフ
- **サイト別統計**: サイトごとのアクセス試行回数、ブロック回数、使用時間
- 設定ページの「統計」タブに表示

---

## データソース

既存のDailyStatsを集計するだけなので、新たなデータ収集は不要。

| 指標 | ソース |
|------|--------|
| ブロック回数 | DailyStats.counts |
| 使用時間 | DailyStats.durations |
| 節約時間 | 推定（ブロック回数 × 平均滞在時間） |
| ストリーク | StreakData |

## 完了条件

- 設定ページに統計タブが表示される
- 週次サマリーが正しく集計される
- サイト別統計が表示される
- `npm run build` / `npm test` が通る
