# モチベーション名言表示

ブロックページにランダムで名言を表示する。
ブロック時のネガティブな体験をポジティブに変える。

---

## 機能

- ブロックページに名言をランダム表示
- 日本語の名言を中心に収録（英語の有名な名言も含む）
- Pro: カスタム名言の追加・ON/OFF設定

### プラン別

| | Free | Pro / Cloud |
|---|---|---|
| デフォルト名言表示 | ✅ | ✅ |
| カスタム名言追加 | ❌ | ✅ |
| 名言ON/OFF | ❌ | ✅ |

---

## デフォルト名言例

- 「集中こそが成功への鍵だ」
- 「今この瞬間に集中しよう」
- 「小さな積み重ねが大きな結果を生む」
- 「未来の自分に感謝されることをしよう」
- 「やるべきことに戻ろう。あなたならできる」

20〜30個を `src/lib/quotes.ts` に収録。
