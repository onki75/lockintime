# UI設計 概要

## 画面一覧

MVPで必要な5画面。Pencil.devでデザイン → React + Tailwind CSS v4で実装。

| # | 画面 | ファイル | サイズ | 詳細 |
|---|------|---------|--------|------|
| 1 | [設定ページ](./options.md) | `src/options/Options.tsx` | フルページ | メインの管理画面 |
| 2 | [ポップアップ](./popup.md) | `src/popup/Popup.tsx` | 360px幅 | クイック操作 |
| 3 | [ブロックページ](./blocked.md) | `src/blocked/Blocked.tsx` | フルページ | リダイレクト先 |
| 4 | [オンボーディング](./onboarding.md) | `src/options/Options.tsx` 内 | フルページ | 初回ウィザード |
| 5 | 共通ダイアログ | コンポーネント | モーダル | サイト追加・プリセット選択 |

## 実装順序

1. **設定ページ** — 全機能のメインUI。ここでコンポーネント（トグル、リスト、ダイアログ等）を作る
2. **ポップアップ** — 設定ページのコンポーネントを再利用して軽量版を構築
3. **ブロックページ** — シンプルなメッセージ表示 + 一時解除ボタン
4. **オンボーディング** — 設定ページのクエリパラメータで切替（`?onboarding=true`）

## デザインシステム

### カラー

| 用途 | カラー | Tailwind |
|------|--------|----------|
| プライマリ | #2563EB | `blue-600` |
| 成功/有効 | #16A34A | `green-600` |
| 危険/ブロック | #DC2626 | `red-600` |
| 警告 | #F59E0B | `amber-500` |
| 背景 | #F9FAFB | `gray-50` |
| カード背景 | #FFFFFF | `white` |
| テキスト | #111827 | `gray-900` |
| サブテキスト | #6B7280 | `gray-500` |
| ボーダー | #E5E7EB | `gray-200` |

### タイポグラフィ

- フォント: system-ui, -apple-system, 'Hiragino Sans', 'Noto Sans JP', sans-serif
- 見出し: `text-xl font-bold`（設定ページ）、`text-lg font-bold`（ポップアップ）
- 本文: `text-sm`（14px）
- サブテキスト: `text-xs text-gray-500`

### コンポーネント

| コンポーネント | 用途 | 使用画面 |
|--------------|------|---------|
| Toggle | ルールON/OFF、アダルトフィルター | 設定、ポップアップ |
| RuleCard | ルール1件の表示（アイコン+ドメイン+制限タイプ+操作） | 設定、ポップアップ |
| GroupCard | グループの表示（展開可能） | 設定 |
| Button | 各種ボタン | 全画面 |
| Dialog | モーダルダイアログ | 設定、ポップアップ |
| PresetPicker | プリセット選択UI（Free制限付き） | 設定、ポップアップ、オンボーディング |
| Badge | 制限タイプ表示（完全ブロック、時刻制限等） | 設定、ポップアップ |

### アイコン（Lucide React）

| アイコン | 用途 |
|---------|------|
| `Shield` | ブロック中 |
| `ShieldOff` | ブロックページ |
| `Clock` | 時刻制限 |
| `Folder` | グループ |
| `Plus` | 追加 |
| `Trash2` | 削除 |
| `Settings` | 設定リンク |
| `ChevronDown` | 展開/折りたたみ |
