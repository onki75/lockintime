LockInTime プライバシーポリシー

最終更新日: 2026-05-31

LockInTime（以下「本拡張機能」）は、ユーザーの集中を支援するサイトブロッカー兼スクリーンタイム管理ツールです。本拡張機能は、機能の提供に必要な最小限のデータのみを扱い、原則としてユーザーのデータを端末内に保存します。

1. 端末内に保存されるデータ（外部送信されません）

以下のデータはすべてお使いのブラウザのローカルストレージ（chrome.storage.local）に保存され、外部サーバーには送信されません。

  • ブロック対象サイト・グループの設定、制限タイプ（フルブロック、時間帯、1日の回数・利用時間、クールダウン、遅延、位置情報）
  • スクリーンタイムの記録、連続達成日数（ストリーク）、各種統計
  • オンボーディングの進捗、アプリ内の設定状態

2. 位置情報

位置情報ベースの制限ルール（特定の場所でのみブロックする等）を設定した場合に限り、ブラウザの位置情報 API を使用して現在地を取得します。取得した座標は端末内でのルール判定にのみ使用し、外部サーバーには送信・保存しません。

3. 外部に送信されるデータ

有料プランの確認・購入を行う場合に限り、以下のデータを当方の決済関連サーバーへ送信します。

  • ライセンス確認時: メールアドレス
  • 購入手続き時: メールアドレス、選択したプラン・課金間隔

決済処理は Stripe を通じて行われます。クレジットカード情報など決済情報の入力・保管は Stripe 上で行われ、本拡張機能がカード情報を取得・保存することはありません。Stripe のプライバシーポリシーは https://stripe.com/privacy をご参照ください。

無料で利用する範囲では、メールアドレスを含むいかなる個人データも外部に送信されません。

4. 第三者への提供・トラッキング

本拡張機能は、広告、アクセス解析、行動トラッキングのための第三者サービスを使用しません。閲覧履歴やページ内容を収集・送信することはありません。データを第三者に販売することはありません。

5. データの保存期間と削除

端末内のデータは、ユーザーが拡張機能をアンインストールするか、ブラウザのデータを削除することでいつでも消去できます。決済に関連して送信されたメールアドレスは、ライセンス管理の目的で必要な期間のみ保持します。削除をご希望の場合は下記までご連絡ください。
