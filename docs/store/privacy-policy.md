# LockInTime プライバシーポリシー

> このファイルはプライバシーポリシーの**原稿**です。Chrome ウェブストアは「公開 URL」を要求するため、これを Web ページとして公開し（例: GitHub Pages、Notion 公開ページ、自社サイト）、その URL を Developer Dashboard に登録する。公開後、最終更新日とお問い合わせ先を実値に置き換えること。

最終更新日: 2026-05-31（公開時に更新）

## 1. 基本方針
LockInTime（以下「本拡張機能」）は、ユーザーの集中を支援するサイトブロッカー兼スクリーンタイム管理ツールです。本拡張機能は、機能の提供に必要な最小限のデータのみを扱い、原則としてユーザーのデータを端末内に保存します。

## 2. 端末内に保存されるデータ（外部送信されません）
以下のデータはすべてお使いのブラウザのローカルストレージ（chrome.storage.local）に保存され、外部サーバーには送信されません。

- ブロック対象サイト・グループの設定、制限タイプ（フルブロック、時間帯、1日の回数・利用時間、クールダウン、遅延、位置情報）
- スクリーンタイムの記録、連続達成日数（ストリーク）、各種統計
- オンボーディングの進捗、アプリ内の設定状態

## 3. 位置情報
位置情報ベースの制限ルール（特定の場所でのみブロックする等）を設定した場合に限り、ブラウザの位置情報 API を使用して現在地を取得します。取得した座標は端末内でのルール判定にのみ使用し、外部サーバーには送信・保存しません。

## 4. 外部へのデータ送信
本拡張機能は、ユーザーのデータを外部のサーバーに送信しません。すべての処理はお使いの端末内（ブラウザ内）で完結します。アカウント登録は不要で、メールアドレスなどの個人情報を収集することもありません。

## 5. 第三者への提供・トラッキング
本拡張機能は、広告、アクセス解析、行動トラッキングのための第三者サービスを使用しません。閲覧履歴やページ内容を収集・送信することはありません。データを第三者に販売・提供することはありません。

## 6. データの保存期間と削除
端末内のデータは、ユーザーが拡張機能をアンインストールするか、ブラウザのデータを削除することでいつでも消去できます。本拡張機能は外部にデータを保持しません。
