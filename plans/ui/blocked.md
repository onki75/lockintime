# ブロックページ

ブロック対象サイトにアクセスした際のリダイレクト先。

---

## レイアウト

```
┌──────────────────────────────────────────────────────────┐
│                                                          │
│                    🛡（ShieldOff）                        │
│                                                          │
│            このサイトはブロック中です                       │
│                 youtube.com                              │
│                                                          │
│         アクセスが制限されています                          │
│                                                          │
│    「未来の自分に感謝されることをしよう」                    │ ← 名言（ランダム）
│                                                          │
│    [⏱ 5分だけアクセスする]    [⚙ 設定を開く]              │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

---

## コンポーネント分割

```
src/blocked/
├── Blocked.tsx          # ルート
├── components/
│   ├── BlockMessage.tsx  # アイコン + メッセージ + ドメイン
│   ├── Quote.tsx         # 名言表示
│   └── Actions.tsx       # 一時解除 + 設定リンク
```
