# ポップアップ

Chrome拡張アイコンクリック時に表示。360px幅のコンパクトUI。

---

## レイアウト

```
┌──────────────────────────────────┐
│ 🛡 LockInTime          🎉 5日残 │ ← トライアルバッジ（期間中のみ）
├──────────────────────────────────┤
│ ブロック中: 8サイト               │
├──────────────────────────────────┤
│ 📁 SNS (6)     完全    [ON/OFF] │
│ ⛔ youtube.com  完全    [ON/OFF] │
│ 🕐 x.com    平日9-18   [ON/OFF] │
│ ⛔ nicovideo.jp 完全    [ON/OFF] │
├──────────────────────────────────┤
│ 🔞 アダルトフィルター    [ON/OFF] │
├──────────────────────────────────┤
│ [+ サイトを追加]                  │
│ [プリセットから追加]               │
├──────────────────────────────────┤
│                   [⚙ 設定を開く] │
└──────────────────────────────────┘
```

---

## コンポーネント分割

```
src/popup/
├── Popup.tsx              # ルート
├── components/
│   ├── PopupHeader.tsx    # ロゴ + トライアルバッジ
│   ├── StatusBar.tsx      # ブロック中: N件
│   ├── RuleList.tsx       # ルール一覧（コンパクト版）
│   ├── QuickActions.tsx   # サイト追加 + プリセット + 設定リンク
│   └── AddSiteQuick.tsx   # クイックサイト追加（ドメイン入力のみ）
```

設定ページの共通コンポーネント（Toggle, Badge）を再利用。
