## プラン詳細

### Free（無料）
ログイン不要で使える基本プラン。

- 完全ブロック + 使用時刻制限（曜日別スケジュール含む）
- ブロックルール: **5件まで**
- プリセットから追加可能
- ブロックページ表示

**目的**: まず使ってもらい、価値を体験してもらう。

### Pro（月額300円 / 年額2,400円）
全制限タイプが使える標準プラン。

- **全7制限タイプ解放**
- ブロックルール: **無制限**
- ストリーク機能（ヒートマップ + 数値）
- 優先サポート

**目的**: 柔軟な制限設定を求めるパワーユーザー向け。

### Pro Lifetime（買い切り 4,800円）
Proの永久ライセンス。

- Proと同じ全機能
- 今後追加されるPro機能も含む
- サブスク疲れのユーザー向け

**価格設定根拠**: Pro月額の16ヶ月分。1年4ヶ月以上使うならお得。

### Cloud（月額500円 / 年額4,800円）
Pro + クラウド同期。

- Pro全機能に加えて:
- **デバイス間リアルタイム同期**（ブロックリスト・ストリーク・DailyStats・表示設定）
- **クラウドバックアップ**（データ消失防止）
- Googleアカウントでログイン

**目的**: 複数PCを使うユーザー向け。Firebaseの運用コストをカバー。

### Cloud Lifetime（買い切り 9,800円）
Cloudの永久ライセンス。

- Cloudと同じ全機能
- 今後追加されるCloud機能も含む

**価格設定根拠**: Cloud月額の約20ヶ月分。1年8ヶ月以上使うならお得。

## 決済

### Chrome Web Store決済
- Chrome Web Store の Payments API を使用
- Googleアカウントと自動連携
- サブスク管理はChrome Web Storeが担当

### 代替: Stripe
- Chrome Web Store決済が使えない場合のフォールバック
- Webページでの決済 → ライセンスキー発行 → 拡張機能に入力

## フェーズとの対応

| フェーズ | プランへの影響 |
|----------|---------------|
| MVP | Free機能を実装。プラン制限（5件上限）の仕組みを入れる |
| P1 | Pro機能を実装（全制限タイプ）。課金の導線を入れる |
| P2 | Pro機能を拡充（位置情報・ストリーク） |
| P3 | Cloud機能を実装（クラウド同期） |

### MVPでの認証の扱い
- **Freeユーザーはログイン不要**で使える
- Proへのアップグレード導線を表示（「5件を超えるにはProプランが必要です」等）
- Googleログインの仕組みはP1で実装（Pro課金開始時）
